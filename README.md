# Tarea: Backend - Django y DRF

## Descripción
Este proyecto consiste en la creación de una API REST utilizando Django y Django REST Framework (DRF).Se implementó un sistema de gestión de Usuarios y Roles siguiendo la arquitectura MTV.

## Requisito Especial
Siguiendo la consigna de la tarea, se empleó la conexión a la base de datos **SQLite** en lugar de PostgreSQL.

## Endpoints Principales
* `http://127.0.0.1:8000/api/users`
* `http://127.0.0.1:8000/api/roles`
* `http://127.0.0.1:8000/admin` (Panel de administración)
